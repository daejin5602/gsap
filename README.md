# gsap
🌿 GSAP Tutorial Playground

GSAP(GreenSock Animation Platform)을 공부하면서 정리한 실습 레포입니다.
하나씩 따라 만들면서 애니메이션 감각 + 코드 흐름을 익히는 것을 목표로 하고 있습니다.

📌 What’s Inside

기본 Tween 실습

Timeline으로 애니메이션 순서 제어

ScrollTrigger로 스크롤 기반 인터랙션 구현

작은 UI/UX 애니메이션 실험

📁 Folder Structure
/01-basic         # GSAP 기본 문법 연습
/02-timeline      # Timeline 애니메이션
/03-scroll        # ScrollTrigger 실습
/playground       # 자유로운 GSAP 실험 공간

🛠 Tech Stack

GSAP 3

HTML / CSS / JavaScript

(Optional) Vite / Parcel 등 빌드 툴 실험 예정

🎯 Goals

프론트엔드 애니메이션 감각 익히기

UI 요소에 자연스러운 동작 넣기
